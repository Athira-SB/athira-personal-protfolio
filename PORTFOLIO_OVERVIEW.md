## 📱 Responsive Breakpoints

### Mobile (< 640px)
- Full width layout
- Hamburger menu in navbar
- Single column sections
- Larger touch targets
- Stack all cards vertically

### Tablet (640px - 1024px)
- Two-column layouts
- Hamburger menu closes
- Show more items per row
- Optimized padding

### Desktop (> 1024px)
- Multi-column layouts
- Full navigation visible
- Grid layouts (2-4 columns)
- Maximum content width
- Hover effects enabled

## 🎨 Color Theme System

### Color Variables (CSS)
```
--color-primary          → Main brand color
--color-primary-dark     → Darker shade (hover)
--color-primary-light    → Lighter shade (background)

Usage: rgb(var(--color-primary))
```

### 5 Built-in Themes
1. **Blue** - Professional, trustworthy
2. **Purple** - Creative, innovative
3. **Green** - Growth, sustainability
4. **Orange** - Energy, enthusiasm
5. **Pink** - Modern, friendly

### Dark Mode
- Toggle dark/light manually
- Automatically adjusts all colors
- Persists theme preference
- Works with all theme colors
